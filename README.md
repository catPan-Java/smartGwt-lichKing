# smartGwt-lichKing
基于smartGwt企业级Ajax 开发框架的扩展，整合spring、Gilead4Gwt（hibernate）、gwtent（反射）、Highcharts等，通过实体类注解来定义各种模型。扩展组件使用、数据绑定也就两行代码，还有那么多smartGwt原生组件，你们专心写业务逻辑吧……传说中的开发神器！
